# TaskFlow

TaskFlow is a study project built to practice Flutter, GetX, REST API integration, Git, Clean Code, project architecture and C#.

## Tech Stack

- Flutter
- Dart
- GetX
- C#
- ASP.NET Core Web API
- Entity Framework Core
- SQLite

## Goal

The goal of this project is to create a small full-stack task management app with a Flutter mobile client and a C# REST API.
